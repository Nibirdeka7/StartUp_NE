# Startup NorthEast 

**StartUp_NE** is a premier innovation ecosystem platform dedicated to catalyzing the startup renaissance in Northeast India. Founded in December 2025, the platform serves as a strategic bridge connecting visionary entrepreneurs with resources, capital, and a collaborative network to build sustainable, high impact ventures.

## Key Features

* **Startup Directory**: A comprehensive, searchable database of verified startups across all eight Northeastern states.
* **Founder Ecosystem**: Integrated tools for entrepreneurs to list their ventures, manage profiles, and connect with partners.
* **Service Marketplace**: Access to strategic execution, market intelligence, and operational support through a specialized service provider network.
* **Dynamic Blog Platform**: A full-featured content engine for ecosystem updates and thought leadership, supporting Markdown and media.
* **Admin Governance**: Secure dashboard for moderators to review, approve, and manage the platform's ecosystem data.
* **PWA Ready**: Built with Progressive Web App capabilities for a seamless mobile experience.

## Technology Stack

### Frontend
* **React 19 & Vite**: Utilizing the latest React features with high-performance development tooling.
* **Tailwind CSS 4**: Advanced utility-first styling with modern animation support.
* **Framer Motion**: Powering fluid, high-end UI transitions and interactions.
* **Lucide React & Tabler Icons**: A comprehensive set of professional iconography.

### Backend & Infrastructure
* **Supabase**: Provides robust Authentication, PostgreSQL database, and real-time state management.
* **Cloudinary**: Handles high-performance media storage and delivery for startup assets and user content.
* **Transactional Messaging**: Integrated with EmailJS for automated communications and approval workflows.
